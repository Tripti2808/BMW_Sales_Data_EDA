# BMW Sales Data EDA (2010-2024)

We have worked on  a BMW car sales data set from 2010-2024 where we have analysed the data and get the data insights after manipulating and transforming the data as per our requirement.
We have manipulated data by changing the data types and getting the Price in INR from USD, calculating the total sales and replacing the categorical values to numerical in order to get more better insights.

We have shown the insights in the form of visuals like bar graphs, pie charts and horizontal bar graphs.
